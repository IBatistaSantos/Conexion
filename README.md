## Descrição

Construção de uma API para criação e gerenciamento de Deals (negócios) com integração do catálogo de produtos e pedidos realizados na Vtex. Oportunizando uma melhor estratégia de marketing para revenda.


## Tecnologia e Metodologia

-Node/Typescript
- Nestjs
- Bull
- Jest
- Nodemailer
- Prisma ORM
- TDD


## Instalação

```bash
$ npm install
```

## Executando a API

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Teste

```bash
# unit tests
$ npm run test


# test coverage
$ npm run test:cov
```
